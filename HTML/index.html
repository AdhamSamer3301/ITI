HTML :

It's an abbreviation for Hyper Text Markup Language.
It can't be considered as a Programming language. It contains tags.
Each tag contains attributes and each attribute contains name and value.

The Tags :

The main tags are Head and Body.
The Head tag contains:

  1. meta :

    2. It consists of :

    • charset:
      • An international unicode that allows the browser to understand every language written in the webpage correctly. It's "utf-8".
    • name:
      • A name for the webpage that make it easier to understand the content of the webpage.
    • content:
      • We write here the description of the webpage to make it easier for the search engine to find the website and be relevant for search.

  2. title :

    3. A title for the webpage shown to the user and in the tabs.

  3. link :

    4. It's a link for other files, like a css file for the coloring and beautiful UI.

  4. script :

    5. A tag the allows me to use another code in another file and apply it in my HTML code, like using a JavaScript file to make the webste more responsive and interactive for the user.
